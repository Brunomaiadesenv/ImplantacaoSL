⚙️ Configuração do Firebase

📌 Como configurar:

1. Acesse console.firebase.google.com
2. Crie um projeto → Realtime Database → Criar banco de dados (modo teste)
3. Vá em Configurações do Projeto → Seus apps → Web → Copie o firebaseConfig
4. Cole os valores abaixo e clique em Salvar

🚀 Publicação – GitHub Pages (gratuito)

Passos para publicar gratuitamente:

1. Crie um repositório no github.com (ex: supernova-implantacao)
2. Faça upload do arquivo index.html para o repositório
3. Vá em Settings → Pages → Source: main branch / root
4. Seu sistema estará em: https://seu-usuario.github.io/supernova-implantacao

✅ Gratuito, sem limite de acessos, HTTPS automático.
Alternativa: arraste o arquivo para tiiny.host ou netlify.com/drop
